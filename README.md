## 本站宗旨

「禅屋」的宗旨是想藉此现代网络平台，让全球有缘的华人了解：禅是一门真正有智慧的修行者必须亲自实证的、不可思议的「形而上的心灵科学」。

因为，禅所要实证的是宇宙生命是如何存在与运作的真相，这是佛陀在菩提树下悟道时所发现的。这里所谓的「智慧」，是《力庄严三昧经》所说：「离心、意、识诸境界，故名不可思议智。」所谓的「心灵」，是《金刚经》所说：「如来说诸心，皆为非心，是名为心。」也是法禅师父所说：「这本心或本性既非物质也非精神；既非心也非物。...那是个绝对体性。在这绝对体性中，精神与物质是一体的。」

事实上，今日科学家所发现的量子世界，就是最接近这不可思议、「非心」的心灵世界。这就是为什么六祖会说：「此法门是最上乘，为大智人说，为上根人说。小根小智人闻，心生不信。」
